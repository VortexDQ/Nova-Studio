## Editing

| Premiere | Nova | Status |
|----------|------|--------|
| Ripple trim | Trim tool **Ripple (R)** + edge drag | **Done** |
| Roll trim | Trim tool **Roll (N)** + drag cut point | **Done** |
| Slip trim | Trim tool **Slip (Y)** + horizontal drag | **Done** |
| Slide trim | Trim tool **Slide (U)** + drag between neighbors | **Done** |
| Selection trim | Trim tool **Selection (V)** + edge drag | **Done** |
| Text-based editing | — | **Plugin** |
| Generative Extend | — | **Planned** |
| Multicam | — | **Planned** |

### Trim tool shortcuts

| Key | Tool |
|-----|------|
| V | Selection |
| R | Ripple |
| N | Roll |
| Y | Slip |
| U | Slide |

Use the **Trim** dropdown on the timeline transport bar or **Edit → Trim tool** menu.
